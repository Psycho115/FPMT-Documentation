# 材料显示颜色 `COLOR`

---

## 输入格式

> $N_{mat}$, matName, `COLOR`, $r$, $g$, $b$

| 参数              | 说明                    |
| ----------------- | ----------------------- |
| `KEYWORD` = `COLOR` |                         |
| $r$, $g$, $b$     | RGB色彩空间内的颜色参数 |


## 示例

```c
1, mat1, TYPE, ...
1, mat1, COLOR, 163, 163, 173
```
