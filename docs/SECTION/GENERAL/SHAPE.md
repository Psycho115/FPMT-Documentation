# 截面形状 `SHAPE`

---

## 输入格式

> $N_{sect}$, sectName, `SHAPE`, $P_1$, $P_2$...

| 截面参数            | 参数说明 |
| ------------------- | -------- |
| `KEYWORD` = `SHAPE` |          |
| $P_1$, $P_2$ ...    | 形状参数 |

**前置条件**

- [截面种类 `TYPE`](/SECTION/GENERAL/TYPE.md) 已定义
