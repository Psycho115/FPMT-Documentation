# 实常数 REALCONSTANT

---

## 通用输入格式

 /REALCONSTANT/
$N_{real}$, realName, `KEYWORD`, ...
\#\#

| 参数       | 说明                                  |
| ---------- | ------------------------------------- |
| $N_{real}$ | 实常数编号, 从1开始, 必须**连续编号** |
| realName   | 实常数名称, 自定义, **不能重名**      |
| `KEYWORD`  | 参数关键词                            |
