# 质量惯性矩 `INER`

---

## 输入格式

### 集中质量 `MASS`

> $N_{real}$, realName, `MASS`, $DOF_i$, $m_i$...

| 参数               | 说明                  |
| ------------------ | --------------------- |
| `KEYWORD` = `MASS` |                       |
| $DOF_i$            | 自由度标识符，取3分量 |
| $m_i$              | 对应自由度的质量分量  |


### 集中惯性矩 `MOMENT`

> $N_{real}$, realName, `MOMENT`, $DOF_i$, $I_i$...

| 参数                 | 说明                  |
| -------------------- | --------------------- |
| `KEYWORD` = `MOMENT` |                       |
| $DOF_i$              | 自由度标识符，取3分量 |
| $m_i$                | 应自由度的惯性矩分量  |

---

## 有效单元种类

- [质量惯性矩单元 `MASS`]()

---

## 示例

```c
1, real1, TYPE, INER
1, real1, MASS, 1, 10, 2, 10, 3, 10
1, real1, MOMENT, 3, 5e-2
```
