# 节点 ```/PARTICLE/```

---

## 节点定义

> /PARTICLE/<br>
$N_{particle}$, $x$, $y$, $z$<br>
\#\#<br>

| 参数           | 说明                                                |
| -------------- | --------------------------------------------------- |
| $N_{particle}$ | 节点序号, 从 1 开始编号, 可不连续编号，建议连续编号 |
| $x$, $y$, $z$  | 节点初始坐标                                        |


---

## 示例

```c
/NODE/
1, 0.00, 0, 0
2, 1.00, 0, 0
3, 2.00, 0, 0
4, 3.00, 0, 0
5, 4.00, 0, 0
6, 5.00, 0, 0
##
```
